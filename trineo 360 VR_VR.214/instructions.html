Instructions for sharing KeyShotVR

Below is your KeyShotVR. To view the KeyShotVR by itself open the trineo 360 VR_VR.214.html file in the trineo 360 VR_VR.214 folder.

Uploading to your web site:

  • Inside your trineo 360 VR_VR.214 folder, upload the "files" folder and trineo 360 VR_VR.214.html file to a publicly accessible server.
  • In your webpage where you would like your KeyShotVR to be viewed from, create an <iframe> tag in the <body> section of the html file and update the <src> attribute to reflect the location of the VR on the webserver. Example: <iframe src="trineo 360 VR_VR.214.html" width="1201" height="803" frameborder="0"></iframe>
  • The trineo 360 VR_VR.214.html file must be kept in the same directory as the "files" folder.

If you are sharing via email. You will need to send the entire zipped trineo 360 VR_VR.214 folder to the end recipient.
